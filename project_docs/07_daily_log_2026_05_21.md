## Important Findings

### Exposure Grain Is Valid

`order_id + order_item_id + seller_id + product_id` 在当前数据中唯一。

### Joins Are Reliable

核心连接关系健康：

```text
order_items -> orders: 100%
order_items -> products: 100%
order_items -> sellers: 100%
orders -> customers: 100%
products -> translation: 99.96%
```

只有商品重量/尺寸相关字段存在极少缺失：

```text

product_length_cm: 0.016%
product_height_cm: 0.016%
product_width_cm: 0.016%
```
